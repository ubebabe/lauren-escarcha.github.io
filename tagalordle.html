description

Tagalog = a dialect from the Philippines
Wordle = a daily word puzzle
Tagalog + Wordle = Tagalordle

Tagalordle is inspired by the global phenomenon, Wordle , with some Tagalog language learning for English speakers.
As an avid Wordle player, this project was the perfect opprotunity to not only gain skills in website development,
but helped me to become more fluent in my family's language.
Now, I can share this approachable and accessible tool to anyone who is interested in learning a bit more about a new language :)

how to play

  1. read the Tagalog vocabulary at the top
  2. type any 5-letter English word for its corresponding definition
  3. continue to guess the definition based on previous attempts
  4. if the correct definition is found before the 6 attempts are up, the game is done!
